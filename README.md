## .i pilno tadji

.i jarco tu'a lo tarji cnino liste bu'u la'o zoi [GitLab Releases](https://gitlab.com/qq542vev/gahiste/-/releases) zoi .i benji le liste noi ke'a mapti la'o zoi slabu IME zoi ku'o ko

## .i lidne lerfu

.i ci lo liste klesi cu zasti tu'a ru

 * .i se cmene be zoi zoi `*-at-*.txt` zoi zo'u
   * zoi zoi `@` zoi lidne lerfu
   * mu'u la'o zoi `@bau Ōćä @bangu` zoi
 * .i se cmene be zoi zoi `*-bs-*.txt` zoi zo'u
   * zoi zoi `\` zoi lidne lerfu
   * mu'u la'o zoi `\bau Ōćä \bangu` zoi
 * .i se cmene be zoi zoi `*-em-*.txt` zoi zo'u
   * zoi zoi `!` zoi lidne lerfu
   * mu'u la'o zoi `!bau Ōćä !bangu` zoi
 * .i se cmene be zoi zoi `*-zi'o-*.txt` zoi zo'u
   * noda lidne lerfu
   * mu'u la'o zoi `bau Ōćä bangu` zoi
